## Codebase Patterns
- Use `{{<tgimg>}}` shortcode for images with src, width, caption parameters
- Use `{{<fields>}}` and `{{<field "Name">}}description{{</field>}}` for field definition tables
- Use `{{% pageinfo %}}` for callout boxes at top of pages
- Use `{{<alert title="..." color="...">}}` for inline alerts (info, warning)
- Screenshots are stored co-located in the same directory as the markdown file
- Front matter uses YAML with: title, linkTitle, weight, description, tags, categories
- Section landing pages use _index.md, content pages use index.md
- Run `npm i` before `hugo --minify` for production builds
